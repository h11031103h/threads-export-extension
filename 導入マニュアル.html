すれっしゅ 導入マニュアル

他社様へ拡張機能を共有する際の、インストールと利用手順です。

1. 概要

すれっしゅは、Meta の SNS「Threads」のタイムライン上で表示されているポストを収集し、CSV ファイルとして保存する Chrome 拡張機能です。

  • 検索結果やタイムラインをスクロールすると、表示されたポストを自動で収集
  • 「純粋ポストのみ」で引用・リプライを除外可能
  • 収集したデータを CSV でダウンロード（ポスト本文・URL・投稿者・日時など）

2. 動作環境

項目内容
ブラウザGoogle Chrome（推奨：最新版）
OSWindows / macOS / Chrome OS など Chrome が動作する環境
対象サイトThreads（threads.net / threads.com）
インストール方式開発者モードによる「パッケージ化されていない拡張機能」の読み込み

※ Chrome 以外のブラウザでは動作しません。

3. 共有するファイル

他社へ渡すときは、拡張機能のフォルダ一式をそのまま ZIP にまとめて共有してください。

3.1 フォルダ名

  • threads-export-extension という名前のフォルダごと渡すか、ZIP の解凍後にこの名前になるようにしてください。

ZIP で渡す場合の例（Windows）

  • threads-export-extension フォルダを右クリック → 「送る」→「圧縮 (zip) 形式フォルダー」
  • できた threads-export-extension.zip をメール添付や共有フォルダで送付

  • ZIP を解凍し、中身の threads-export-extension フォルダを、わかりやすい場所（例：デスクトップ、ドキュメント）に置く

4. インストール手順

4.1 Chrome で拡張機能のページを開く

  1. Google Chrome を起動する
  2. アドレスバーに chrome://extensions と入力して Enter
  3. 「拡張機能」の管理画面が開く

4.2 デベロッパーモードを有効にする

  1. 画面右上の 「デベロッパーモード」 のスイッチを ON にする
  2. スイッチが青くなり、下に「パッケージ化されていない拡張機能を読み込む」などのボタンが表示されれば OK

4.3 拡張機能を読み込む

  1. 「パッケージ化されていない拡張機能を読み込む」 をクリック
  2. フォルダ選択のダイアログが開く
  3. threads-export-extension フォルダを選ぶ（フォルダの中身ではなく、threads-export-extension というフォルダ自体を選択）
  4. 「フォルダの選択」（または「選択」）をクリック

4.4 インストール完了の確認

  • 一覧に 「すれっしゅ」 が表示され、スイッチが ON になっていればインストール完了です
  • アイコン（虫眼鏡で「ス」を覗いているマーク）が表示されていれば問題ありません

5. 使い方

5.1 パネルを表示する

  1. Threads のサイトを開く（例: https://www.threads.net/search または https://www.threads.com）
  2. ログインしている状態で、検索結果やタイムラインが表示されるページにする
  3. 画面右上に 「すれっしゅ」 のパネル（黒い枠）が表示される

5.2 ポストを収集する

  1. 「純粋ポストのみ収集」 … 引用・リプライを除外したいときだけチェックを入れる
  2. ページを下にスクロールして、取得したいポストを画面上に表示する
  3. パネルの 「収集: ○ 件」 の数字が増えていけば、収集できています

5.3 CSV をダウンロードする

  1. パネルの 「Download CSV」 ボタンをクリック
  2. 新しいタブ（すれっしゅのダウンロード用ページ）が開く
  3. 画面に表示されている ファイル名（例: threads_posts_20260223_162350.csv） をクリック
  4. 「名前を付けて保存」 のダイアログが開くので、保存場所を選んで「保存」をクリック
  5. 指定したファイル名で CSV が保存されます

5.4 CSV の内容

保存された CSV には、おおよそ次の列が含まれます。

列名説明
id連番
post_textポストの本文
post_urlポストの URL
author_username投稿者の @ユーザー名
posted_at投稿日時

6. トラブルシューティング

パネルが表示されない

  • Threads のページ（threads.net / threads.com）を開いているか確認する
  • いったん Threads のタブを閉じ、もう一度開き直す
  • chrome://extensions で「すれっしゅ」が ON になっているか確認する

「収集: 0 件」のまま増えない

  • ページをスクロールして、ポストが画面に表示されているか確認する
  • 検索結果やタイムラインがまだ読み込まれていない場合は、少し待ってからスクロールする

ダウンロード用タブが開かない / エラーが出る

  • パネルに表示される赤いエラー文を確認する
  • 拡張機能を一度削除し、再度「パッケージ化されていない拡張機能を読み込む」で同じフォルダを読み込む
  • Chrome を最新版に更新する

更新したのに古い動作のまま

  1. Threads のタブをすべて閉じる
  2. chrome://extensions で「すれっしゅ」の ⟳ 更新 をクリック
  3. もう一度 Threads のページを開く

7. 注意事項

  • 取得しているのは、ブラウザの画面上に表示されている公開情報のみです
  • Threads の利用規約に従ってご利用ください
  • 拡張機能の配布・改変を行う場合は、ライセンス（README 等）を確認してください
