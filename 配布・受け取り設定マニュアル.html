すれっしゅ GitHubリンク受け取り設定マニュアル

このマニュアルは、Chrome Web Storeには公開せず、GitHubの公開リンクから すれっしゅ を受け取って使うための手順です。

最終更新日: 2026-06-28

1. 受け取り用リンク

GitHub公開ページ:

https://github.com/h11031103h/threads-export-extension

直接ZIPをダウンロードするリンク:

https://github.com/h11031103h/threads-export-extension/archive/refs/heads/main.zip

2. GitHubからダウンロードする

  1. ChromeでGitHub公開ページを開く
  2. 緑色の Code ボタンを押す
  3. Download ZIP を押す
  4. threads-export-extension-main.zip がダウンロードされる

3. ZIPを解凍する

  1. ダウンロードしたZIPを解凍する
  2. threads-export-extension-main フォルダを確認する
  3. 削除しない場所へ移動する

4. Chromeで読み込む

  1. Chromeで chrome://extensions を開く
  2. 右上の デベロッパーモード をONにする
  3. パッケージ化されていない拡張機能を読み込む を押す
  4. 解凍した threads-export-extension-main フォルダ自体を選ぶ
  5. 一覧に すれっしゅ が表示され、ONになっていれば完了

5. 使い方

  1. ChromeでThreadsにログインする
  2. 検索結果またはタイムラインを開く
  3. すれっしゅパネルの スタート を押す
  4. 収集したい範囲までスクロールする
  5. Download CSV を押してCSVを保存する

6. 更新版を受け取る場合

  1. GitHubページを開く
  2. Code → Download ZIP で新しいZIPをダウンロードする
  3. ZIPを解凍する
  4. 古いフォルダを新しいフォルダで置き換える
  5. chrome://extensions で すれっしゅ の 更新 を押す
  6. Threadsを開き直す

7. トラブルシューティング

症状確認すること
GitHubページが見られないリンク、リポジトリ公開状態、ネットワーク制限を確認する
Chromeに読み込めないZIPを解凍済みか、manifest.jsonがフォルダ直下にあるか確認する
パネルが表示されないThreadsのURL、拡張機能がONか、フォルダを移動していないかを確認する
収集: 0件のままスタートを押したか、ページをスクロールしたかを確認する
